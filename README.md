# SeleniumMiniProject
Automated Testing Project.
Ky projekt është zhvilluar duke përdorur Selenium WebDriver për automatizimin e testimeve në një aplikacion web. Qëllimi i projektit është të demonstrojë përdorimin praktik të automatizimit të testimit.

---

##  Qëllimi i Projektit

- Automatizimi i skenarëve bazë të testimit  
- Verifikimi i funksionaliteteve kryesore të një aplikacioni web  
- Përdorimi i locator-ëve (ID, Name, XPath, CSS Selector)  

---

##  Teknologjitë e Përdorura

- Java  
- Selenium WebDriver  
- TestNG / JUnit  
- Maven  
- ChromeDriver  

---

##  Funksionalitetet

- Hapja automatike e browser-it  
- Navigimi në faqe web  
- Plotësimi i formave  
- Klikimi i butonave  
- Verifikimi i rezultateve (Assertions)  
- Mbyllja e browser-it  

---

##  Si të ekzekutohet

1. Klono repository-n
2. Hape projektin në IDE (IntelliJ/Eclipse)
3. Ekzekuto klasën e testimit
