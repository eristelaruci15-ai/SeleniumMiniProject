How to set the project up:

1. Download the folder.zip
2. Folders:base, pages, tests, utils have already been created. [the path:Selenium_Project\src\test\java\com\tealiumdemo\tests]
3. The codes have already been made at base folder (BaseTest), pages folder (HomePage, RegisterPage) and utils folder (TestDataGenerator)
4. You MUST do the project in VS Code NOT InteliJ, otherwise it will show a lot of errors and won't work.
5. Add the project folder in vs code. 
6. Download the needed extension about java, to make the code work.
7. You also need to download chromedriver.exe and add it to the folder of the project so the whole project works and the website opens.
8. Each test should be added at the tests folder [Selenium_Project\src\test\java\com\tealiumdemo\tests]
